# the font configs

## Linux

Download from `https://www.nerdfonts.com/font-downloads`.
(I suggest to install JetBrainsMono Nerd Font Mono)

Cd this folder, then run:

```bash
mkdir -p ~/.local/share/fonts
cp ./*.ttf ~/.local/share/fonts/
fc-cache -fv
```

Restart the terminal, set the font of the configs of terminal.

## Win

Win 不用专门下载字体，但是终端字体设置比较神奇，在左边。
