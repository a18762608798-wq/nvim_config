# file tree

## project operation

* **换到一个没开的文件**：`<space><space>`
* **在已经开的几个文件里切**：`Shift-h` / `Shift-l`
* **在文件树和正文之间跳**：`Ctrl-h` / `Ctrl-l`
* 折叠某个目录：`L`
* 退回上一个目录：`backspace`
* 把当前目录作为根目录：`.`
* 关闭当前文件缓冲区：`<space>bd`
* show the hidden files: `H`

## create and move the files

* Create a file: `a`
* Create a folder: add a `\` at the end.
* Move the files or folder: `m` then write the new file name;
or `tab` to choose and `m` at the targeted folder.
* Copy the folder, tab then c
