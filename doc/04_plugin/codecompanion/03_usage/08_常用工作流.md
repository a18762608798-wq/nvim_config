# 常用工作流

### 解释当前文件

```text
#{buffer}
Explain this file. Focus on the public API and the data flow.
```

### 解释 selection

```text
#{selection}
Explain this selected code line by line.
```

### 生成 commit message

```text
#{diff}
Generate a concise Conventional Commit message.
```

或者：

```text
@{get_changed_files} 查看当前 git diff，并生成 Conventional Commit message。
```

### Review 当前改动

```text
@{get_changed_files} Review the current diff. Focus on bugs, edge cases, and missing tests.
```

### 查找某个符号的所有用法

```text
Use @{grep_search} to find all usages of `RandomMeasAdd`, then summarize how it is used.
```

### 理解项目结构

```text
@{files} 找到项目里和 RandomMeasAdd 有关的文件，读取关键文件，并解释整体结构。
```

### 让 agent 读、改、测

```text
@{agent} 修复当前模块的问题。先查看相关文件和 diagnostics，再修改文件，最后运行相关测试。
```

### 分析终端报错

```text
#{terminal}
刚才命令失败了，帮我分析原因，并给出下一步排查命令。
```

如果希望它继续执行命令：

```text
@{run_command} 根据上面的错误，运行一个安全的诊断命令来确认问题。
```

### 分析 Neovim 报错

```text
#{messages}
刚才 nvim 报错了，帮我分析原因。
```

### 搜外部代码风格

```text
@{web_search} 搜索 RandomMeas.jl 库的代码注释风格，并总结可模仿的规范。
```

或者固定 URL：

```text
/fetch https://github.com/bvermersch/RandomMeas.jl/blob/89c492bfb05508e5babe9c8c2c40697995be9e42/src/Estimators.jl#L227-L238
```
