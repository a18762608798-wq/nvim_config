# 推荐默认使用策略

普通问答：

```text
#{buffer}
Explain this file.
```

局部修改：

```text
#{selection}
Rewrite this code to be clearer.
```

需要搜项目：

```text
@{files} ...
```

需要 agent 式完成任务：

```text
@{agent} ...
```

需要运行命令：

```text
@{run_command} ...
```

需要看 git diff：

```text
#{diff}
```

或：

```text
@{get_changed_files}
```

需要外部网页：

```text
/fetch <url>
```

需要联网搜索：

```text
@{web_search}
```
