# 核心概念：`#`、`@`、`/` 的区别

| 语法       | 名称                  | 作用                                 | 谁主动执行                       | 典型用途                                              |
| -------- | ------------------- | ---------------------------------- | --------------------------- | ------------------------------------------------- |
| `#{...}` | Editor Context      | 把 Neovim 当前状态作为上下文发给 LLM           | 你发送消息时 CodeCompanion 注入     | 当前 buffer、selection、git diff、messages、terminal 输出 |
| `@{...}` | Tools / Tool Groups | 让 LLM 可以调用工具读取、搜索、修改、运行命令          | LLM 决定是否调用，CodeCompanion 执行 | 搜文件、读文件、改文件、跑命令、看 diagnostics                     |
| `/...`   | Slash Commands      | 手动把额外内容插入 chat buffer，或执行 chat 内命令 | 你手动触发                       | 选 buffer/file、fetch URL、compact history、加图片       |

记忆方式：

```text
# = 给模型看什么
@ = 允许模型做什么
/ = 我手动插入/执行什么
```
