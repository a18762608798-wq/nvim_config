# usage

## keymaps

```text
Quarto Preview：

<leader>qv    预览 Quarto


Quarto 发送当前 cell：

<leader>qj    当前 cell 发到 Julia REPL
<leader>qp    当前 cell 发到 Python REPL


REPL 管理：

<leader>jr    打开/聚焦 Julia REPL
<leader>jh    隐藏 Julia REPL
<leader>jR    重启 Julia REPL

<leader>pr    打开/聚焦 Python REPL
<leader>ph    隐藏 Python REPL
<leader>pR    重启 Python REPL
```
